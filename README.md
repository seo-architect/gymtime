# Gymtime PWA

Simple mobile-first workout timer PWA.

## Run locally

Use the tiny dev command:

```bash
make dev
```

Optional custom port:

```bash
make dev PORT=5173
```

Direct Python command (no Make required):

```bash
python3 -m http.server 8000
```

Then open:

- `http://localhost:8000`

## Install on phone (Add to Home Screen)

iOS (Safari):

1. Open the site in Safari.
2. Tap Share.
3. Tap Add to Home Screen.
4. Tap Add.

Android (Chrome):

1. Open the site in Chrome.
2. Tap the 3-dot menu.
3. Tap Add to Home screen or Install app.
4. Confirm Add/Install.
